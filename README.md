# Datenstrukturen

Dieses Maven-Projekt sammelt einfache Implementierungen klassischer Datenstrukturen:

- Stack (LIFO)
- Singly Linked List
- Queue
- B-Baum
- Adjazenzmatrix-Graph
- Adjazenzlisten-Graph
- Binary Search Tree
- Skip List

Die Unit-Tests demonstrieren typische Operationen und können mit Maven ausgeführt werden:

```bash
mvn test
```
